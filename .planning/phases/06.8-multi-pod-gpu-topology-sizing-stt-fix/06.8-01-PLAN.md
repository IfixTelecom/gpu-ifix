Prepare the STT fix at image/script level (fix iii from RESEARCH): regenerate the whisper weights tarball in HF-hub-cache layout and point the speaches child at it via HF_HUB_CACHE. The current tarball extracts files FLAT into `/weights/whisper` (invisible to the scan; no README.md). Fix: tarball in cache layout + `HF_HUB_CACHE=/weights/whisper`.

Purpose: Make `/v1/audio/transcriptions` resolvable on every topology (it has been broken since 06.7 UAT). This blocks STT on the 2×3090 shape too.
Output: Modified `upload-weights.sh` (whisper block) + `supervisord.conf` ([program:speaches] env). Image rebuild + tarball re-upload happen in Plan 02 (operator + live validation).
