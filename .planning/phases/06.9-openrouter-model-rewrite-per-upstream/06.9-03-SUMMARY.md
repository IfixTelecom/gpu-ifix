## Decisions Made

- **R5 honesty: 3-pass JSON sequence is honest, not a defect.** The OpenRouter director's body rewrite path is `RewriteJSONModel → injectProviderOrder → injectStreamOptionsIncludeUsage`. Each pass unmarshals + mutates + marshals. The godoc explicitly says "3 sequential JSON unmarshal+marshal passes" — the pre-revision "single rewrap pass" language was DELETED. A future refactor MAY combine the passes into a single helper, but only if profiling identifies allocation pressure. The current shape is clearer to reason about per-pass.
- **WhisperAbortGuard exported (capital W).** Task 2 originally named the function `whisperAbortGuard` (unexported). Task 4 needed to wrap from the cmd/gateway package and exported it as `WhisperAbortGuard`. The rename was applied in-place; tests still pass.
- **NewResolverForTesting in production package, not in _test.go.** The proxy package tests need a fixture-backed resolver. The models package's existing `newResolverFromMap` is unexported. Options were: (a) add `export_test.go` in models — but that only exposes to `models_test` package, not `proxy_test`; (b) make `newResolverFromMap` exported — but that pollutes the production package's exported API; (c) create a separate `testing.go` file with the helper. Option (c) was chosen because it documents the helper as test-only via godoc and stays in the production package (Go convention — see `testing/iotest` etc.).
- **Director defensively forwards original body on duplicate-model statusCode=400.** The WhisperAbortGuard wrapper SHOULD intercept the duplicate-model case BEFORE the proxy runs. But if for any reason the wrapper isn't in the handler chain (e.g. scaffold test variant), the director's switch case handles statusCode=400 by forwarding the ORIGINAL body unchanged + emitting a WARN log. This keeps the director's behavior deterministic — never silently sends a malformed body, never 500s.
- **canonicalAliasForUpstream is private to proxy package (not in models).** Only the Whisper director needs this map (the multipart "missing model field" injection path). The OpenRouter and Embed directors detect missing model in the JSON body via `m["model"]` presence and the resolver-with-empty-alias passthrough. Keeping the map local to where it's used keeps the resolver's scope clean and the tier-1 onboarding requirement (add an entry here when registering a new STT upstream) explicit.
- **Body's incoming alias is read in the Embed director.** Pre-06.9 the director just wrote `text-embedding-3-small` literal regardless of what the client sent. Post-06.9 the director reads `body.model` first, then calls `resolver.Resolve(alias, "openai-embed")`. If `body.model` is absent, the alias is empty string and the resolver returns "" (passthrough); OpenAI will reject the request as missing model. That's the right semantics — clients MUST send a model field for embeddings.
- **R12 log emission only when log != nil.** The Director constructors accept `log *slog.Logger`; production wiring always passes the real logger, but tests use `discardLogger()`. The directors guard each log call with `if log != nil` so that unit tests with nil loggers (if any are added in the future) don't NPE.
- **Boot order migrate→resolver preserved by inspection.** Plan asked to verify migrate (line 172, `db.Up`) runs before resolver init (line 252, `models.NewResolver` + `resolver.Refresh`). No reorder needed.

```go
func BuildOpenRouterDirector(
    upstream *url.URL,
    authBearer string,
    providerOrder []string,
    allowFallbacks bool,
    resolver *models.Resolver,
    upstreamName string,
    log *slog.Logger,
) func(*http.Request) {
    base := BuildDirector(upstream)
    return func(r *http.Request) {
        base(r)
        if authBearer != "" {
            r.Header.Set("Authorization", "Bearer "+authBearer)
        }
        if r.URL.Path != "/v1/chat/completions" { return }
        if r.Body == nil { return }
        body, err := io.ReadAll(r.Body)
        _ = r.Body.Close()
        if err != nil || len(body) == 0 {
            r.Body = io.NopCloser(bytes.NewReader(body))
            r.ContentLength = int64(len(body))
            return
        }

        // PASS 1 — model rewrite via resolver (Phase 06.9 OR-FIX).
        rewritten, _, err1 := models.RewriteJSONModel(body, resolver, upstreamName)
        if err1 != nil {
            if log != nil {
                log.Debug("openrouter_director model rewrite parse failed",
                    "upstream", upstreamName,
                    "error_class", fmt.Sprintf("%T", err1),
                )
            }
            rewritten = body
        }

        // PASS 2 — provider.order injection (D-C2).
        patched, err2 := injectProviderOrder(rewritten, providerOrder, allowFallbacks)
        if err2 != nil {
            if log != nil {
                log.Warn("openrouter_director injectProviderOrder failed",
                    "upstream", upstreamName,
                    "error_class", fmt.Sprintf("%T", err2),
                )
            }
            rewriteRequestBody(r, rewritten)
            return
        }

        // PASS 3 — stream_options.include_usage injection (Pitfall 5).
        patched = injectStreamOptionsIncludeUsage(patched)

        r.Body = io.NopCloser(bytes.NewReader(patched))
        r.ContentLength = int64(len(patched))
        r.Header.Set("Content-Length", strconv.Itoa(len(patched)))
    }
}
```

### Whisper rewriter helper (full source — R6 edge cases)

```go
func rewriteMultipartModelViaResolver(
    body []byte,
    contentType string,
    resolver *models.Resolver,
    upstreamName string,
) ([]byte, string, int, error) {
    mediaType, params, err := mime.ParseMediaType(contentType)
    if err != nil {
        return nil, "", 0, err
    }
    if !strings.HasPrefix(mediaType, "multipart/") {
        return nil, "", 0, fmt.Errorf("content-type is not multipart: %s", mediaType)
    }
    boundary, ok := params["boundary"]
    if !ok || boundary == "" {
        return nil, "", 0, fmt.Errorf("missing boundary in multipart content-type")
    }

    mr := multipart.NewReader(bytes.NewReader(body), boundary)
    var out bytes.Buffer
    w := multipart.NewWriter(&out)

    modelSeen := false
    for {
        part, perr := mr.NextPart()
        if perr == io.EOF { break }
        if perr != nil { return nil, "", 0, perr }

        if part.FormName() == "model" {
            if modelSeen {
                // R6 / WARNING-3 — duplicate "model" form field abort.
                _, _ = io.Copy(io.Discard, part)
                _ = part.Close()
                return nil, "", http.StatusBadRequest, nil
            }
            modelSeen = true

            aliasBytes, rerr := io.ReadAll(part)
            _ = part.Close()
            if rerr != nil { return nil, "", 0, rerr }
            alias := string(aliasBytes)
            target := resolver.Resolve(alias, upstreamName)

            fw, werr := w.CreatePart(part.Header)
            if werr != nil { return nil, "", 0, werr }
            if _, werr := fw.Write([]byte(target)); werr != nil {
                return nil, "", 0, werr
            }
            continue
        }

        // Non-model part: stream through binary-safe.
        fw, werr := w.CreatePart(part.Header)
        if werr != nil { return nil, "", 0, werr }
        if _, werr := io.Copy(fw, part); werr != nil {
            return nil, "", 0, werr
        }
        _ = part.Close()
    }

    // R6 — missing model field: inject via canonicalAliasForUpstream.
    if !modelSeen {
        canonAlias := canonicalAliasForUpstream[upstreamName]
        target := resolver.Resolve(canonAlias, upstreamName)
        fw, werr := w.CreateFormField("model")
        if werr != nil { return nil, "", 0, werr }
        if _, werr := fw.Write([]byte(target)); werr != nil {
            return nil, "", 0, werr
        }
    }

    if cerr := w.Close(); cerr != nil { return nil, "", 0, cerr }
    return out.Bytes(), w.FormDataContentType(), 0, nil
}
```

```go
func WhisperAbortGuard(inner http.Handler, resolver *models.Resolver, upstreamName string, log *slog.Logger) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        ct := r.Header.Get("Content-Type")
        if !strings.HasPrefix(ct, "multipart/form-data") || r.Body == nil {
            inner.ServeHTTP(w, r)
            return
        }
        body, err := io.ReadAll(r.Body)
        _ = r.Body.Close()
        if err != nil {
            r.Body = io.NopCloser(bytes.NewReader(nil))
            inner.ServeHTTP(w, r)
            return
        }
        _, _, statusCode, _ := rewriteMultipartModelViaResolver(body, ct, resolver, upstreamName)
        // Restore body so the inner proxy (director) can re-read.
        r.Body = io.NopCloser(bytes.NewReader(body))
        r.ContentLength = int64(len(body))

        if statusCode == http.StatusBadRequest {
            if log != nil {
                log.Warn("whisper_abort_guard rejected duplicate model field",
                    "upstream", upstreamName,
                )
            }
            w.Header().Set("Content-Type", "application/json")
            w.WriteHeader(http.StatusBadRequest)
            _, _ = w.Write([]byte(whisperDuplicateModelMessage))
            return
        }
        inner.ServeHTTP(w, r)
    })
}
```
